# 요구사항


## 포인트
- 포인트충전-회원이 포인트충전요청을 하면 회원의포인트를 충전시킵니다.
- 포인트충전실패-회원이 아닌경우는 포인트를 충전할수 없습니다.
- 포인트사용-회원이 포인트사용요청을 하면 회원의포인트를 사용합니다.
- 포인트사용실패-잔액이 부족한경우는 포인트사용을 할 수 없습니다.


## 주문
상품주문-사용자가 상품주문요청을 하면 상품주문을 합니다.
